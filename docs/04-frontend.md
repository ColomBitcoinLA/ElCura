# 04. Frontend (HTML/CSS/JS)

## Principios
- CSS con variables y secciones (reset, tipografía, layout, componentes, utilidades).
- Animaciones con `transform` y `opacity` para rendimiento.
- Grid/Flexbox para diseño responsivo.

## Componentes clave
1. **Header:** logo + navegación.
2. **Hero:** portada/ilustración + mensaje.
3. **Secciones destacadas:** obras, autor, comunidad.
4. **Footer:** contacto + redes.

## Experiencia "Libro Real"
- Diseño 3D con `transform: perspective()`.
- Transiciones de página con CSS y un estado JS mínimo.
- En móviles, fallback a desplazamiento vertical por páginas.

## Accesibilidad
- Tamaños de fuente escalables.
- Contraste AA.
- Navegación por teclado en lectores.
