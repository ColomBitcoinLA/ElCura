# El Cura: Cuentos y Poesías

Proyecto para construir una plataforma moderna, segura y escalable para la marca "El Cura" (cuentos y poesías). La meta es habilitar lectura en línea, audiolibros, video, notas/segmentos resaltados y una experiencia de libro realista con animaciones ligeras.
