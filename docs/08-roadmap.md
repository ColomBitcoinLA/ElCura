# 08. Roadmap y escalamiento

## Fase 1: MVP
- Frontend estático + API básica.
- Lectura y progreso guardado.
- Admin simple para cargar libros.

## Fase 2: Suscripciones
- Integración de pagos.
- Descargas para suscriptores vitalicios.

## Fase 3: IA
- Recomendaciones de lectura.
- Resumen de capítulos.
- Asistente para navegar libros.

## Fase 4: Escalamiento
- CDN + cache.
- Microservicios (opcional).
